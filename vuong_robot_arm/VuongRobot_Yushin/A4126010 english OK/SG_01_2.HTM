Input signals from IMM
 Bkwd Frwd
Signal name Description
Bad Part Signal This signal turns ON when the injection molding machine produces defective product during full-auto operation. When this signal turns ON, the take-out robot performs flip motion at the reject discharge position, before it discharges the product.
IMM full-auto This signal turns ON when the injection molding machine starts full-auto operation. During Auto operation, take-out motion will not start unless this signal is ON.
