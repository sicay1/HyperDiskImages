Speeds for product release motion
œS11 Traverse
This is the speed at which the take-out robot moves to the product release side after product take-out and ascent.

œS14 Release side descent
This is the speed at which the take-out robot moves to the product release position.

œS16 Release side ascent
This is the speed at which the take-out robot ascents from the release position.

œS17 Traverse return
This is the speed at which the take-out robot moves to the mold side after product release.
