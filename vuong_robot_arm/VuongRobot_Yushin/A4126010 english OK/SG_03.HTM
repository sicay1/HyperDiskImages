Output signals to IMM
 Bkwd Frwd
Signal name Discription
Mold closed
interlock		 This is a permission signal for mold closed motion.
When this signal is ON, the injrction molding machine can perform mold closed motion.
Mold open
interlock		 This is a permission signal for mold open motion.
When this signal is ON, the injection molding machine can perform mold open motion.
Cycle start During Auto operation, this signal indicates to the injection molding machine that take-out motion has been completed. when this signal turns ON, the injection molding machine performs mold closed motion and next cycle starts.
When a take-out failure occurs, this signal does not turn ON and the injection molding machine does not perform mold closed motion.
Ejector order This is a permission signal for eject motion of the ejector.
