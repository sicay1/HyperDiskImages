Timers 2 for product take-out motion
œT05 Ejector delay
 This timer waits the time from when the take-out robot starts take-out descent motion until the eiector order is outputted. This timer is only enabled when the ejector interlock is set to Interlock 1 or Interlock 2 during operation mode setting.

œT06 Main arm kick return waiting
 This timer waits the time from when the product chuck closes until the kick return motion starts.

œT08 Main arm ascent waiting
 This timer waits the time for the main arm to start ascending from the kick return position.
