Positions for take-out motion
Åú Approach waiting position
The end-of-arm tool (Runner chuck) waits at this position near to the mold.
Åú Descent position
The mold opens and the end-of-arm tool descents into the mold area from this position. Set this position so that when the end-of-arm tool (runner chuck) descents, it does not interfere with the mold.
Åú Take-out position
After the end-of-arm tool descents into the mold area, the take-out robot takes out the product and/or runner from this position.
Åú Kick return position
Set this position so that when the end-of-arm tool ascent after chucking of a product and/or runner, the product and/or runner does not interfere with the mold.
