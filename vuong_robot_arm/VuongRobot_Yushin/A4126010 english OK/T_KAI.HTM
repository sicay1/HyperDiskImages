Timers for product release motion
œT14 Release side descent
This timer waits the time from traverse to the release side until descent to the product release position.

œT15 Pre-product release
This timer waits the time from descent to the product release position until product release.

œT16 Post-product release
This timer waits the time from product release at the product release position until the end-of-arm tool ascent.
